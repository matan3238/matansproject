## הערות

- כל התמונות הן מ-Unsplash - שירות תמונות חינמי ופתוח, או מחנויות היצרנים (Apple Store, Samsung)
- התמונות מותאמות לגודל 500x500 פיקסלים או בגודל מלא מהמקור
- אם תמונה לא נטענת, הקוד מציג תמונת placeholder אוטומטית
- ניתן להחליף את התמונות בכתובות אחרות לפי הצורך

## הוספת תמונות חדשות

כדי להוסיף תמונות חדשות למוצרים:

1. מצא תמונה מתאימה ב-Unsplash או שירות אחר
2. העתק את כתובת התמונה
3. עדכן את השדה `image` במוצר ב-`models/Product.js` או ב-`scripts/seed.js`

### דוגמה:
```javascript
{
  title: 'מוצר חדש',
  image: 'https://images.unsplash.com/photo-XXXXX?w=500&h=500&fit=crop',
  // ... שאר הפרטים
}
```
